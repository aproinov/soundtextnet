## Apa itu WiFi?
WiFi adalah jaringan nirkabel yang memungkinkan perangkat elektronik seperti smartphone, tablet, komputer, printer dan lain sebagainya untuk bisa terhubung dengan internet tanpa kabel dengan menggunakan sinyal frekuensi radio.

Pada saat ini WiFi sudah menjadi kebutuhan setiap orang terutama di perkantoran, restoran, mal dan fasilitas publik lainnya. Hanya saja untuk mengaksesnya biasanya dibutuhkan kata sandi atau password WiFi agar bisa terhubung.

Berikut ada beberapa cara untuk melihat password WiFi di hp atau PC yang sudah terhubung namun lupa atau tidak mengingat password tersebut. Dengan cara ini, kamu bisa kembali terhubung WiFi dengan mudah dan cepat.

## Cara Melihat Password WiFi di HP
Agar bisa terhubung dengan internet melalui WiFi tanpa harus memasukkan password, maka bisa mempelajari cara melihat password WiFi di bawah ini. Kamu bisa terhubung tanpa perlu repot mencari tahu saat lupa password WiFi.

## 1. WiFi Master Key
Cara untuk melihat password Wifi di HP menggunakan aplikasi WiFi Master Key, aplikasi ini bisa diakses secara gratis dan bisa menampilkan password WiFi yang pernah tersimpan. Adapun cara menggunakannya sebagai berikut:

1. Download aplikasinya di Play Store
2. Buka aplikasi WiFi Master Key
3. Pilih nama WiFi untuk mengetahui password
4. Pilih periksa keamanan, aplikasi akan melakukan scan
5. Setelah scan selesai, akan ada informasi password WiFi

## 2. WiFi Key Recovery
Adapun cara untuk melihat password WiFi di hp yang pernah terhubung tapi lupa, maka bisa menggunakan aplikasi WiFi Key Recovery. Adapun cara menggunakannya sebagai berikut:

1. Download aplikasinya di Play Store
2. Buka aplikasi WiFi Key Recovery
3. Jika meminta izin akses root, silahkan pilih izinkan
4. Secara otomatis muncul informasi password WiFi

## 3. Free WiFi Password Recovery
Aplikasi untuk melihat password WiFi di hp ini mudah digunakan dan gratis, hanya saja untuk mengaksesnya perlu izin root. Untuk lebih jelasnya, berikut cara menggunakannya.

1. Download aplikasi di Play Store
2. Buka aplikasi Free WiFi Password Recovery
3. Pilih menu Grant permission & recovery password
4. Kemudian izinkan akses root pada hp
5. Maka akan muncul informasi SSID dan password WiFi

## 4. WiFi Password Viewer
Untuk melihat password WiFi di hp yang pernah tersimpan dan terhubung selanjutnya bisa memakai aplikasi WiFi Password Viewer. Adapun cara menggunakannya sebagai berikut:

1. Download aplikasi di Play Store
2. Buka aplikasi WiFi Password Viewer
3. Kemudian izinkan akses root pada hp
4. Otomatis muncul informasi SSID dan password WiFi

## 5. WiFi Test
Sebagai alternatif untuk menghubungkan internet ke WiFi di hp yang terkunci password bisa melalui aplikasi WiFi Test secara gratis. Apapun cata menggunakannya sebagai berikut:

1. Download aplikasi di Play Store
2. Buka aplikasi WiFi Test yang terpasang di hp
3. Pilih nama WiFi yang ingin diketahui passwordnya
4. Pilih bertuliskan WiFi yang Support, klik OK
5. Jika proses berhasil, muncul tulisan Koneksi Sukses
6. Dengan menggunakan aplikasi diatas, maka bisa melihat kata sandi WiFi di hp dengan mudah sehingga bisa terkoneksi internet secara gratis. Hanya saja butuh akses root agar aplikasi bisa berjalan dengan lancar. Yuk, coba sekarang!
